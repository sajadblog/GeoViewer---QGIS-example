# GeoViewer --- QGIS-example
Is a simple project from the QGIS library that can show and present data on Geographic maps.

Map Layer Type :

- Raster
- Vector
- WMS
- XYZ

Drawing Items : 

- Simple Marker Item
- Marker or Flag Item
- Multi Position Item
- Sector Item
- Text Item
- Colorize Path Item

Graphical Tools :

- Measure Tool
- Multi Position Drawing Tool
